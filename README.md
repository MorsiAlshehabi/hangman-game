# 🎮 Galgje (Hangman Game)

Een interactieve Hangman-game gemaakt met PHP, HTML en CSS.

---

## 📌 Over het project

Dit is een klassiek Hangman-spel waarin de speler een woord moet raden door letters te kiezen.  
Het spel bevat categorieën en ook een optie om zelf een woord te kiezen.

---

## 🚀 Functionaliteiten

- 🎯 Willekeurige woorden per categorie
- ✍️ Mogelijkheid om zelf een woord te kiezen
- ⌨️ Interactief toetsenbord (A-Z)
- ❤️ Aantal pogingen (chances)
- 🖼️ Hangman afbeeldingen die veranderen per fout
- 🎉 Win- en verlies schermen
- 🔁 Reset functie om opnieuw te spelen
- 📂 Woorden worden geladen uit `words.txt`

---

## 🖥️ Startscherm

De startpagina bevat twee opties:

- 🎲 Speel met een random woord / categorie
- ✍️ Kies zelf een woord

Play with a random word
OR
Choose your own word
